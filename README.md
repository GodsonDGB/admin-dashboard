# Dashboard Project

## Introduction

This project is part of the **Intermediate HTML and CSS course** from **The Odin Project**.
The goal of this project is to practice building a complete dashboard layout using **CSS Grid** as the main layout tool.

By completing this project, I strengthened my understanding of **grid layouts, nested grids, layout structuring, and UI composition**.
---

## Project Objective

The objective of this project is to build a **dashboard interface** that includes:

- A **sidebar** for navigation
- A **header** containing search and user information
- A **main content area** displaying:
  - Projects
  - Announcements
  - Trending users

The layout is primarily built using **CSS Grid**, with nested grid containers for organizing different sections of the interface.

---

## Technologies Used

- **HTML5**
- **CSS3**
- **CSS Grid**
- **Google Fonts**
- **Material Design Icons (SVG)**

---

## Features

- Dashboard layout built with **CSS Grid**
- **Nested grid containers** for complex layouts
- Sidebar navigation with icons
- Header with:
  - Search bar
  - User profile
  - Action buttons

- Main content displaying:
  - Project cards
  - Announcements
  - Trending section

- Clean and structured UI
